Render molecules as inline SVG in web by RDkit

  • 2018-06-06
  • Birdlet

There are two kinds of images: bitmap and vetorgram in the computer. The vectorgram is very suitable for illustrating chemical structures which can be easily drawled by edge and text.

The RDKit package can draw chemical structures in bitmap or vactorgram with only several codes. Sometime we want to dynamically render high quality figures of molecules in web, and the SVG format is the best choice to do this.
